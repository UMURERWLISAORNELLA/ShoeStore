
# Gradient Store - Premium E-commerce Platform

## Project Overview

Gradient Store is a modern, responsive e-commerce platform built with React and TypeScript. It offers a premium shopping experience with a beautiful gradient-themed design.

## Features

- **Product Catalog**: Browse shoes for Men, Women, and Kids
- **User Authentication**: Secure login and registration system
- **Shopping Cart**: Add items, manage quantities, and checkout
- **Order Management**: Track your order history
- **Admin Dashboard**: Manage products and orders (admin users)
- **Responsive Design**: Optimized for all device sizes
- **Search & Filter**: Find products by category and search terms

## Technologies Used

This project is built with:

- **Frontend**: React 18, TypeScript, Vite
- **UI Components**: shadcn/ui, Tailwind CSS
- **State Management**: React Hooks, TanStack Query
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd gradient-store

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Project Structure

```
src/
├── components/          # Reusable UI components
├── pages/              # Application pages
├── services/           # API services
├── hooks/              # Custom React hooks
└── lib/                # Utility functions
```

## Available Scripts

- `npm run dev` - Start development server with hot-reload
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
